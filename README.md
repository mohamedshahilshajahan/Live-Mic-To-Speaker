## üìå About
**LiveMicToSpeaker** is a lightweight Windows application that routes your microphone input directly to your speakers in real time.  
It‚Äôs available as both a **Portable EXE** and an **Installable EXE**, so you can choose what works best for you.  
- **Portable EXE** ‚Üí Just run it, no installation needed.  
- **Installable EXE** ‚Üí Comes with a setup wizard and creates Start Menu shortcuts.  

No additional requirements for EXE versions ‚Äî works out of the box.  
Python script version requires installing dependencies from `requirements.txt`.

---

## üöÄ Features
- üéô **Real-time microphone to speaker routing**  
- üñ± **Start/Stop from tray icon**  
- ‚è≥ **Auto-stop after idle timeout**  
- ‚å® **Custom global hotkey support**  
- üé® **Tray icon color changes for status**  
- ‚ö° **Low-latency audio**  
- üì¶ **Available as portable or installable**

## üìÇ Files in this Repository
- **Portable EXE** ‚Üí For direct use without installation  
- **Installable EXE** ‚Üí For setup-based installation  
- **Python Script** ‚Üí Requires Python + dependencies
